# M2FS Data Reduction Pipeline


Michigan/Magellan Fiber System (M2FS) reduction pipeline

Steps:
* Bias/Trim/Gain/Dark correction/Cosmic rejection
* Tracing
* Wavelength calibration
* Flatfield
* Sky substraction
* Flux calibration
* Merge observations
